อ้างอิง

เว็บไซต์อ้างอิงที่ใช้ในโครงงาน TPQI Competency Profile System ระบบเทียบเคียงสมรรถนะของบุคคลกับเกณฑ์มาตรฐานสถาบันคุณวุฒิวิชาชีพ (องค์การมหาชน) โดยใช้ Angular 17 พัฒนาเว็บแอปพลิเคชันในฝั่งของ Client และทำการติดต่อกับ Server และใช้ NodeJS ในการเชื่อมต่อจากฝั่ง Client กับ Server โดยใช้ Express.js Framework ช่วยจัดการเรื่องของ APIs หรือการรับส่ง/ข้อมูลที่ส่งไปยัง Web browser และ Database ซึ่ง Database ใช้ MySQL สำหรับเพิ่มประสิทธิภาพกระบวนการจัดเก็บข้อมูลอย่างเป็นระบบ และใช้เเกณฑ์มาตรฐาน สถาบันคุณวุฒิวิชาชีพ (องค์การมหาชน)
